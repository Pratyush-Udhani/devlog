### An application to log anything you want. Daily logs, server logs, app logs, personal logs all in one place. 

### Running the server
- Create a virual env
- `pip install -r requirements.txt`
- `python manage.py runserver`

### CLI Commads: 
- `chmod +x cli/devlog`
- cli/devlog create | show | tag_filter
